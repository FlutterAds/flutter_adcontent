## 💻 支持平台

- 穿山甲-内容输出
- 安卓
- iOS

## 🚀 核心功能

- 🎬 短剧
- 📱 小视频
- 🎮 广告变现
    - 激励视频广告
    - 信息流广告
    - 视频流广告
- 🎉 【独家支持】可 Widget 组件嵌入Flutter 页面
- 🎨 【独家支持】可自定义 UI 样式，与业务完美契合
- 📦 【独家支持】广告事件回调全局打通
